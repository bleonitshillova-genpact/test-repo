## Troubleshooting Quick Fixes

### Port in Use?
```powershell
# Use different port
python -m http.server 3001
```

### Server Not Responding?
- Check terminal - is it running?
- Look for "Server running on..." message

### Postman Error?
- Verify API server terminal is active
- Check URL: http://localhost:8000

### Database Not Found?
```powershell
cd python-tests
python database_test.py
```

## Emergency Commands

**Stop any server:** `Ctrl+C`

**Kill process on port:**
```powershell
# Find process
netstat -ano | findstr :3000
# Kill it (replace PID)
taskkill /PID <PID> /F
```

**Reset Git credentials:**
```powershell
cmdkey /delete:LegacyGeneric:target=git:https://github.com
```
