# Activity 5: Database Connectivity (DBeaver)

## Steps to Demonstrate

### 5.1 Open DBeaver
- Launch DBeaver application

### 5.2 Connect to Database

**Method 1: Create New Connection**
1. Click **Database** → **New Database Connection**
2. Select **SQLite**
3. Click **Next**
4. **Path:** Browse to:
   ```
   C:\Users\602000837\Desktop\test-repo\python-tests\test_database.db
   ```
5. Click **Test Connection** (should show success)
6. Click **Finish**

**Method 2: Open File Directly**
1. **File** → **Open File**
2. Navigate to: `C:\Users\602000837\Desktop\test-repo\python-tests\test_database.db`
3. Click **Open**

### 5.3 View Database Structure

**Navigate in left panel:**
```
test_database.db
  → Tables
    → test_users
      → Columns
        - id (INTEGER, Primary Key)
        - username (TEXT)
        - email (TEXT)
        - created_at (TIMESTAMP)
```

### 5.4 View Data

**Double-click on `test_users` table**

**Expected Data Grid:**
```
ID | Username | Email                  | Created At
---|----------|------------------------|-------------------
1  | alice    | alice@example.com     | 2026-07-06 10:56:20
2  | bob      | bob@example.com       | 2026-07-06 10:56:20
3  | charlie  | charlie@example.com   | 2026-07-06 10:56:20
...
```

### 5.5 Run SQL Queries

**Open SQL Editor (Ctrl+]):**

**Query 1: Select all users**
```sql
SELECT * FROM test_users;
```

**Query 2: Count users**
```sql
SELECT COUNT(*) as total_users FROM test_users;
```
Expected: `6`

**Query 3: Filter by username**
```sql
SELECT * FROM test_users WHERE username = 'alice';
```
Expected: 2 records

**Query 4: Get unique usernames**
```sql
SELECT DISTINCT username FROM test_users;
```
Expected: alice, bob, charlie

### 5.6 Execute Query
- Press **Ctrl+Enter** or click Execute button (▶)
- Results appear in bottom panel

# Activity 6: API Testing with Postman

## Steps to Demonstrate

### 6.1 Start API Server

**Open NEW PowerShell terminal:**
```powershell
cd C:\Users\602000837\Desktop\test-repo\api-server
python server.py
```

**Expected Output:**
```

  GET  http://localhost:8000/users
  GET  http://localhost:8000/users?id=1
  POST http://localhost:8000/users
  GET  http://localhost:8000/status

Press Ctrl+C to stop the server
============================================================
```

### 6.2 Import Collection into Postman

1. **Open Postman**
2. Click **Import** button (top left)
3. Click **Choose Files**
4. Navigate to:
   ```
   C:\Users\602000837\Desktop\test-repo\api-server\postman_collection.json
   ```
5. Click **Open**
6. Click **Import**

**Collection Name:** "Testing Environment API"

### 6.3 Test Each Endpoint

**Test 1: Health Check**
- Click on **"Health Check"** in left panel
- Click **Send** button
- **Expected Response:**
  ```json
  {
    "status": "healthy",
    "timestamp": "2026-07-06T...",
    "uptime": "active"
  }
  ```
- **Expected Status:** `200 OK` (green)

**Test 2: Get All Users**
- Click on **"Get All Users"**
- Click **Send**
- **Expected Response:**
  ```json
  {
    "count": 3,
    "users": [
      {"id": 1, "name": "Alice", "email": "alice@example.com", "role": "Admin"},
      {"id": 2, "name": "Bob", "email": "bob@example.com", "role": "User"},
      {"id": 3, "name": "Charlie", "email": "charlie@example.com", "role": "User"}
    ]
  }
  ```
- **Expected Status:** `200 OK`

**Test 3: Get User by ID**
- Click on **"Get User by ID"**
- Note URL parameter: `?id=1`
- Click **Send**
- **Expected Response:**
  ```json
  {
    "id": 1,
    "name": "Alice",
    "email": "alice@example.com",
    "role": "Admin"
  }
  ```
- **Expected Status:** `200 OK`

**Test 4: Create User (POST)**
- Click on **"Create User"**
- Note: Method is **POST** (orange label)
- Click **Body** tab to view request payload
- Click **Send**
- **Expected Response:**
  ```json
  {
    "message": "User created successfully",
    "user": {
      "id": 4,
      "name": "Diana",
      "email": "diana@example.com",
      "role": "User"
    },
    "timestamp": "2026-07-06T..."
  }
  ```
- **Expected Status:** `201 Created` (green)

**Test 5: Get Status**
- Click on **"Get Status"**
- Click **Send**
- **Expected Response:**
  ```json
  {
    "server": "running",
    "database": "connected",
    "api_version": "1.0",
    "timestamp": "2026-07-06T..."
  }
  ```
- **Expected Status:** `200 OK`

### 6.4 Run All Tests at Once

1. Click on collection name: **"Testing Environment API"**
2. Click **"Run"** button
3. Click **"Run Testing Environment API"**

### Port Already in Use
```powershell

```

### Python Not Found
```powershell
python --version
# or try
python3 --version
```

### API Connection Error in Postman
- Verify server is running: check terminal
- Verify URL: http://localhost:8000
- Check no firewall blocking

### DBeaver Can't Find Database
- Full path: `C:\Users\602000837\Desktop\test-repo\python-tests\test_database.db`
- Run `python database_test.py` first to create it
