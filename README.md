# Testing Environment Setup

This repository demonstrates various testing activities including Git operations, code generation, Python execution, front-end testing, database connectivity, and API testing.

## Activities Covered

1. **Git Operations** - Clone, push, pull, branch management
2. **Code Generation** - Python, JavaScript, and API code
3. **Python Execution** - Script validation and testing
4. **Front-end Application** - Simple web application
5. **Database Connectivity** - DBeaver connection examples
6. **API Testing** - Postman-ready endpoints
